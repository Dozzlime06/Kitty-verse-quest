# Nekomancy

**Nekomancy** is a Play-to-Earn (P2E) adventure game on the Base network, where players collect unique **Nekomancer NFTs**, embark on quests, and build their own cat habitats.  

**Features:**
- Collect and trade unique cat NFTs
- Embark on quests to earn $PAW tokens
- Customize habitats for passive rewards
- Participate in governance and community events
